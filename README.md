# PosturePerfect - TAMUhack 2026 Hardware Track

- Using ultrasonic distance sensor, our device is able to detect and warn the user of bad posture. By measuring the distance of your neck to the chair, and shoulder to the chair, the posture device is able to give an audio warning if the shoulder distace is greater than the spine distance
